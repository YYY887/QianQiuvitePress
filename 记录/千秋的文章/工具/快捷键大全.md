# Typora Linux 快捷键速查表

## 一、文本格式
| 功能 | Linux 快捷键 |
| ---- | ------------ |
| 加粗 | `Ctrl + B` |
| 斜体 | `Ctrl + I` |
| 下划线 | `Ctrl + U` |
| 删除线 | `Alt + Shift + 5` |
| 行内代码 | `Ctrl + Shift + `` |
| 文字高亮 | `Ctrl + Shift + H` |
| 居中对齐 | `Ctrl + Shift + E` |
| 清除格式 | `Ctrl + \` |

## 二、标题操作
- 一级标题：`Ctrl + 1`
- 二级标题：`Ctrl + 2`
- 三级标题：`Ctrl + 3`
- 四级标题：`Ctrl + 4`
- 五级标题：`Ctrl + 5`
- 六级标题：`Ctrl + 6`
- 标题升级：`Ctrl + =`
- 标题降级：`Ctrl + -`

## 三、列表 & 引用
| 功能 | 快捷键 |
| ---- | ---- |
| 无序列表 | `Ctrl + Shift + ]` |
| 有序列表 | `Ctrl + Shift + [` |
| 任务列表 | `Ctrl + Shift + X` |
| 引用块 | `Ctrl + Shift + Q` |
| 向右缩进 | `Tab` |
| 向左缩进 | `Shift + Tab` |

## 四、插入元素
| 功能    | 快捷键                |
| ----- | ------------------ |
| 插入表格  | `Ctrl + T`         |
| 代码块   | `Ctrl + Shift + K` |
| 超链接   | `Ctrl + K`         |
| 插入图片  | `Ctrl + Shift + I` |
| 行内公式  | `Ctrl + Shift + M` |
| 水平分割线 | `Ctrl + Shift + -` |

## 五、视图操作
- 专注/全屏模式：`F11`
- 大纲侧边栏显示/隐藏：`Ctrl + Shift + L`
- 源代码模式切换：`Ctrl + /`
- 切换夜间模式：`Ctrl + Shift + Y`

## 六、文件通用快捷键
- 新建文件：`Ctrl + N`
- 打开文件：`Ctrl + O`
- 保存：`Ctrl + S`
- 另存为：`Ctrl + Shift + S`
- 撤销：`Ctrl + Z`
- 重做：`Ctrl + Y`
- 复制：`Ctrl + C`
- 剪切：`Ctrl + X`
- 粘贴：`Ctrl + V`
- 查找：`Ctrl + F`
- 查找替换：`Ctrl + H`
